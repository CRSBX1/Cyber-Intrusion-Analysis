# Project Description

My data analysis project made for Asia Pacific University's Programming for Data Analysis module assignment, in which I analyzed the relation between the presence of cyber intrusion and TCP handshake duration.
I used UNSW Canberra's UNSW-NB15 Dataset that contains normal and cyber intrusion network traffic data. My analysis utilizes traditional analysis methods, such as statistical analysis and data visualization,
as well as machine learning approaches like Random Forest and XGBoost algorithms.
